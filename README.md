# Login Screen - React + Tailwind

Figma 디자인을 기반으로 구현한 로그인 화면입니다.

## 기술 스택

- React 18
- Vite
- Tailwind CSS

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 프로젝트 구조

```
├── src/
│   ├── components/
│   │   ├── Login.jsx      # 메인 로그인 컴포넌트
│   │   ├── StatusBar.jsx  # 상단 상태바
│   │   ├── SsoIcon.jsx    # Google 아이콘
│   │   └── EyeOffIcon.jsx # 비밀번호 숨김 아이콘
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.js
```
